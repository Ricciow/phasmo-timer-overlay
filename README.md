# Phasmo-Timer-Overlay

Este é um overlay para o jogo Phasmophobia para timers e calculo de velocidade

Aceita funcionalidades simples, configurado apenas para caçadas de duração longa

Keybinds 1-5 para timers
Espaço para contador de passos
R para reset
, e . para alteração de tamanho de mapa
ctrl + esc para fechar o app
